What is N-Queens?

The N-Queens problem asks us to place N queens on an N × N chessboard such that no two queens attack each other.

Why Backtracking?

Instead of checking every possible arrangement, backtracking places queens row by row. Whenever a dead end is reached, it removes the last queen and explores another path.

Hashing Optimization
  • columns[] → occupied columns
  • diag1[] → row + column
  • diag2[] → row − column + N − 1

Safety checking becomes O(1).
